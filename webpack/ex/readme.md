** Passagem de arquivos por referencia pelo webpack

Neste exemplo,   o aquivo index.js passa por referencia
um outro aquivos (duvidaCruel.js) com o conteúdo 
console.log('Sou carregado');
 
 
Ao executar o npm run dev

o console do navegador, mostra o texto 'Sou Carregado?' uma vez que ele foi referenciado 
no arquivo index.js
